## 压缩工具

# MacZip

-   https://ezip.awehunt.com/
-   支持预览的压缩工具

# Keka

-   https://www.keka.io/en/
-   不支持预览的压缩工具

---

# 虚拟机

## VirtualBox

-   优势: 免费
-   劣势: 性能差

## Parallels Desktop

-   优势: 性能利用率高（人话就是机器不差流畅度就不会差）、支持主从间应用融合
-   劣势: 价格贵、每年升级需付费

---

# 翻译软件 Bob

-   https://ripperhe.gitee.io/bob
-   支持划词、截图、输入等翻译方式
-   支持多种翻译源，如百度翻译、有道翻译、阿里翻译等

![bg right:40%](https://files.someok.com/images/2021/11/03/202111031857-L0Wayf.png)

---

# 同步软件 坚果云

-   https://www.jianguoyun.com/
-   支持多目录同步
-   支持增量同步
-   稳定！！！

# 图片视频

---

# iShot 截图/录屏

-   https://apps.apple.com/cn/app/ishot-%E4%BC%98%E7%A7%80%E7%9A%84%E6%88%AA%E5%9B%BE%E5%BD%95%E5%B1%8F%E5%B7%A5%E5%85%B7/id1485844094?mt=12
-   截图后编辑功能比较丰富
-   Mac 上截取窗口支持边角自动透明

---

# uPic 图床

-   https://apps.apple.com/cn/app/upic-%E5%BC%BA%E5%A4%A7%E7%9A%84%E5%9B%BE%E5%BA%8A%E5%B7%A5%E5%85%B7/id1549159979?mt=12
-   支持多种图床
-   起码对于 AWS S3 支持最好
    -   试用过多款图床，能正常上传的还就这个

---

# Pikka 取色器

-   https://apps.apple.com/cn/app/pikka-color-picker/id1195076754?mt=12
-   基本功能都有
-   Library 管理比较方便
-   价格稍贵

---

# OminiConverter

-   https://apps.apple.com/cn/app/omni-converter-%E5%85%A8%E8%83%BD%E8%A7%86%E9%A2%91%E8%BD%AC%E6%8D%A2%E5%99%A8/id1536071582?mt=12
-   转换效率高
-   比直接用 FFMpeg 速度快很多

# JetBrains 家族

# 数据库

## Postgres.app

-   https://postgresapp.com/
-   支持安装多种版本的 PostgresSQL

## DBngin

-   https://dbngin.com/
-   支持安装多个版本的 MySQL、PostgresSQL、MongoDB
-   但是版本有些旧

---

## DBeaver

-   https://dbeaver.io/
-   数据库客户端
-   免费

![w:1150](https://dbeaver.io/wp-content/uploads/2018/03/data_edit.png)

---

# SDKMan

-   https://sdkman.io/
-   支持通过命令行安装多个版本的 jdk，并且方便切换

```
sdk install java
```

---

# Volta

-   https://volta.sh/
-   Node 包管理工具
-   支持安装多个版本的 Node、Npm、Yarn、Pnpm 等
-   比 nvm 快，快很多

---

# Apifox

-   https://www.apifox.cn/
-   免费
-   在 postman 功能的基础上还支持数据测试、Mock 等

![w:1100](https://cdn.apifox.cn/www/screenshot/dark-apifox-api-case-1.png)

---

# Git

-   `JetBrains` 家族软件自带的 Git 功能已足够完善
-   如需管理与切换多个 git 库，可采用 `SourceTree`
